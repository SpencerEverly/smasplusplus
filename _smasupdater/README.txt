The SMASUpdater is a tool to update the episode before launching SMBX2.

To use the updater, put PortableGit (Or download it at https://git-scm.com/download/win ) in the worlds folder, and the bat file in the main SMBX2 folder.

That's all you need to do. Then you can launch it and it'll download SMAS++ for you.

It also has capibilities to run SMBX2.exe afterward when pressing enter, and it shows you if there are any Git errors before launching.

(For v1.0.0, put it into the worlds folder)
